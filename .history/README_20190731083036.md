# count_down

> count_down

## Build Setup
### 安装教程
``` bash
# install dependencies
npm install

# serve with hot reload at localhost:8080
npm run dev

# build for production with minification
npm run build
```
### 使用
> 组件引用
```javascript
import countDown from 'count_time_daydown'

 components:{
    countDown
  }

```
> 全局引用
```javascript
import countDown from 'count_time_daydown'

 components:{
    countDown
  }

```
